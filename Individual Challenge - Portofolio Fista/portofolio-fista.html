Hai Everyone!!!

Perkenalkan saya Fista Puteri Adna biasa dipanggil Fista. Sesuai dengan halaman pertama di Portofolio saya saat ini yakni saya seorang Mahasiswa. Saya berkuliah di Universitas Negeri Semarang mengambil Program Studi Pendidikan Teknik Informatika dan Komputer (S1) yang berada di Fakultas Teknik. Saya merupakan orang yang bermotivasi tinggi dan mempunyai keinginan luas.

Selama saya berkuliah hingga semester 6 ini saya telah mengerjakkan projek yang berhubungan dengan Program Studi saya. Saya telah mengerjakkan project seperti pembuatan website dengan HTML dan CSS dan sebagainya. Dengan begitu saya telah mengembangkan soft skill dan berfokus kepada pembuatan web. Maka saya sangat tertarik dalam pembuatan website yang mempunyai berbagai macam penampilan dan dapat beraction dan menghubungkan dari satu page ke page lain.
